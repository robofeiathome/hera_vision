Instalar as dependências necessárias para o tracking e para a YOLOv8, seguindo os passos abaixo:

    Para o tracking:

        Abra o terminal ou prompt de comando.

        Navegue até a pasta do módulo de tracking (Yolov8_Tracking).

        Execute o comando "pip install -r requirements.txt".

    Para a YOLOv8:

        Abra o terminal ou prompt de comando.

        Navegue até a pasta "yolov8" dentro da pasta do módulo de tracking (Yolov8_Tracking).

        Execute o comando "pip install -r requirements.txt".
        
        
Executar o código track.py

Com esses passos, o pacote estará completo e funcionando

Caso algum erro seja encontrado abrir um issue, por favor. :)
